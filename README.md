# xr手机镜像(MirrorXR)

## 注意

### 本软件开源许可证为**GPL-3.0**，任何人可以免费使用、修改、分发，但请遵守开源许可证的规定


## 简介
本软件可实现手机屏幕镜像到手机应用流转到xr设备

xr设备支持：能够安装第三方软件、基于安卓系统的xr设备

手机支持：能够打开开发者调试(ADB)功能、基于安卓系统的手机
